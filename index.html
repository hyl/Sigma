Sigma

Private web chat using Node.js and WebSocket

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Sigma.

Sigma is the easiest way to create a multi-user, one-on-one chat server using Node.js and WebSocket. Think Omegle, but less horny-teenagery and more open-sourcey. Sigma can be run with the included front end, or just a backend so that you are free to design your own UI. Sigma is super easy to install, with only one dependency:

$ npm install
$ node server.js

You may want to edit server.js before running it so that you can change the port it runs on.
